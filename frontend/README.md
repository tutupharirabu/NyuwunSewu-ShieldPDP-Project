# ShieldPDP Frontend

Enterprise React dashboard for the NyuwunSewu / ShieldPDP FastAPI backend.

## Run

Start the backend on port `8000`, then:

```bash
cd frontend
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

The Vite dev server proxies API requests to `http://localhost:8000`.

Default local credentials:

```text
email: admin@nyuwunsewu.local
password: ChangeMe123!
organization slug: default-organization
```

## Build

```bash
npm run build
```
